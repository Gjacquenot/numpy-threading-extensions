Welcome to fast_numpy_loops’s documentation!¶

Faster loops for NumPy using multithreading and other tricks. The first release will target NumPy binary and unary ufuncs. Eventually we will enable overriding other NumPy functions, and provide an C-based (non-Python) API for extending via third-party functions.

Installation¶

This is a binary package and requires compilation. We recommend using pip or conda to obtain a pre-built version:

$ pip install fast_numpy_loops
# or
$ conda install fast_numpy_loops

To use the package one it is installed:

>>> import fast_numpy_loops
>>> fast_numpy_loops.initialize()

How to use it?¶

Override all the NumPy UFunc loops with multithreaded variants. The original loop will be called via a pool of threads.
